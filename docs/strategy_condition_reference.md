## 一、技術指標庫

對應 `_01_data/` 的指標模組，皆以 OHLCV 為輸入、輸出對應欄位。

### 趨勢類（`indicators_trend.py`）

| 指標 | 函式 | 預設參數 | 用途 |
|------|------|----------|------|
| 簡單移動平均 SMA | `calculate_sma` | window=20 | 趨勢判斷、均線排列基礎 |
| 指數移動平均 EMA | `calculate_ema` | span=20 | 反應較快的趨勢線 |
| MACD | `calculate_macd` | 快 12 / 慢 26 / 訊號 9 | 動能與趨勢轉折 |
| 布林通道 Bollinger | `calculate_bollinger_bands` | window=20、2 倍標準差 | 波動區間、極值參考 |
| 乖離率 BIAS | `calculate_bias` | window=20 | 價格偏離均線的幅度 |

### 動能 / 量能類（`indicators_momentum_volume.py`）

| 指標 | 函式 | 預設參數 | 用途 |
|------|------|----------|------|
| RSI | `calculate_rsi` | period=14 | 超買 / 超賣、動能強弱 |
| KD 隨機指標 | `calculate_kd` | n=9、K 平滑 3、D 平滑 3 | 動能、轉折 |
| 蔡金資金流 CMF | `calculate_cmf` | window=20 | 資金淨流入 / 流出 |
| 能量潮 OBV | `calculate_obv` | — | 量能累積方向 |

### 波動類（`indicators_volatility.py`）

| 指標 | 函式 | 預設參數 | 用途 |
|------|------|----------|------|
| 真實波幅百分比 ATR% | `calculate_atr_pct` | window=14 | 以 % 表示的波動度 |
| 報酬波動率 | `calculate_return_volatility` | window=20、scale=10 | 報酬標準差（縮放後） |
| 唐奇安通道 Donchian | `calculate_donchian` | window=20 | N 日最高 / 最低通道 |

### 型態類（`indicators_pattern.py`）

| 指標 | 函式 | 預設參數 | 用途 |
|------|------|----------|------|
| ZigZag 擺動轉折 | `calculate_zigzag` | pct=2% | 擺動高低點、轉折偵測 |

## 二、均線交叉策略條件

對應 `_02_strategy/ma_strategy/`（單股）與 `_03_multi_strategy/ma_cross/`（多股組合）。
基準策略只有「黃金交叉買、死亡交叉賣」，其餘皆為可疊加的進 / 出場濾網（旗標，預設關）。

### 進場條件（買）

| 條件 | 說明 | 參數 / 常用值 |
|------|------|----------------|
| **黃金交叉**（基準） | 短均線由下往上穿越長均線（短均今天在上、昨天不在） | 短 / 長：50 / 200（可換 21 組） |
| **雙日確認** | 交叉後多等一天，短均**連 2 日**站上長均才進 | — |
| **強 K** | 交叉當日為一根**長紅**（開→收 ≥ 5%）或**跳空**開在短均之上 | 長紅門檻 5% |
| **夾角 > N 度** | 黃金交叉當下，短均與長均的斜率夾角 > 門檻才進（每日 % 斜率當正切換算角度） | 典型 20°；斜率視窗 5 日 |
| **夾角擴大** | 短均的 N 日 % 斜率 > 長均 % 斜率（可要求連 N 日遞增） | 視窗 5 日；遞增日數可設 |
| **盤整過濾（ADX）** | 交叉前一根 K 的 ADX(14) < 門檻才進（趨勢尚未發動） | 門檻 20～25 |
| **多頭排列** | 大方向濾網：**120 日線 > 200 日線 且 收盤站上 120 日線**（固定 120 / 200，不隨交叉期數變動） | — |
| **流動性門檻** | 近 5 日均量 > 門檻才進 | 台股 1,000 張（＝100 萬股） |

### 單均線版額外進場（`single_ma_strategy.py`）

以「收盤上穿單一 MA」為基準的突破型，另有：

| 條件 | 說明 |
|------|------|
| **突破均線**（基準） | 收盤由下往上穿越單一 MA |
| **回踩整理** | 突破前 N 日內，累計達 m 日「當日波動較前日收斂、且收在 MA 下方」才算整理過 |
| **回測支撐反轉** | 突破後回測 MA、再翻上的 pullback-to-MA 進場 |
| 強 K / ADX / 流動性 | 與雙均線版同義，可疊加於突破訊號之上 |

### 出場條件（賣）

| 條件 | 說明 |
|------|------|
| **死亡交叉**（基準） | 短均線由上往下穿越長均線 |
| **跌破短均** | 除死亡交叉外，收盤「由上跌破短均線」也出場 |
| **跌破 MA**（單均線版） | 收盤由上穿越下單一 MA 出場 |

> 多股組合版（`multi_ma_cross.py`）：同樣以黃金交叉買、死亡交叉賣，可選加流動性門檻；差別在多檔共用一份資金。

## 三、參數一覽

| 項目 | 常用值 | 備註 |
|------|--------|------|
| 均線期數（短 / 長） | 50 / 200 | 文章以 21 組短 / 長組合掃描 |
| 強 K 長紅門檻 | 開→收 ≥ 5% | 或跳空開在短均之上 |
| 夾角門檻 | > 20° | 每日 % 斜率當正切；斜率視窗 5 日 |
| ADX 盤整門檻 | < 20～25 | ADX 為 14 期 Wilder |
| 流動性門檻 | 5 日均量 > 1,000 張 | 台股；＝100 萬股 |
| 多頭排列均線 | 120 / 200 | 固定，不隨交叉期數變動 |
| 成交價 | 隔日開盤 | 訊號日收盤判定、隔日開盤成交 |
